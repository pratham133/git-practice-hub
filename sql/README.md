# 🗄️ SQL Track

Beginner-level SQL practice against a small mock e-commerce database.

## Structure

- `exercises/` — sample files used in Git-mechanics practice issues
- `challenges/` — real technical tasks for this track

## Getting Started

Make sure you've read the root [CONTRIBUTING.md](../CONTRIBUTING.md) first.

This track assumes you recieve the same base dataset/files as everyone else in SQL —
do not modify shared files outside the one your issue names.
